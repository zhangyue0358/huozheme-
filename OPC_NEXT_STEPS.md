# 活着吗 OPC 下一步

这份清单是给“不想操作后台”的一人公司用的。你只需要按顺序把账号交互这几步完成，代码和配置我来接。

## 需要你亲自做的事

这些不能由 AI 代替，因为涉及你的账号、手机号、邮箱和服务条款。

1. 注册 Supabase 账号。
2. 创建一个新项目，项目名可以叫 `huozhema`。
3. 打开项目设置里的 API 页面。
4. 把 `Project URL` 和 `publishable key` 发给我。

## 我会接着做的事

拿到两个 key 之后，我会继续完成：

1. 把 `.env` 配好。
2. 运行 `supabase/setup_all.sql` 建表、权限和随笔照片 Storage。
3. 把登录、打卡、随笔、照片、待办从演示数据切换成真实云端数据。
4. 跑通真实登录、打卡、随笔照片、历史日记、待办、好友申请、接受好友、好友今日状态。
5. 打一个 EAS 内部测试包，让手机能安装。

## MVP 先不上复杂功能

第一版只验证一个问题：年轻人会不会愿意每天点一下“我还活着”，并看看朋友有没有出现。

暂时不做：

- 公开广场
- 评论区
- 排行榜
- 复杂待办
- 商业化会员
- AI 陪聊

这些都等真实留存数据出来后再判断。
