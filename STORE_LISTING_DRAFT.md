## 长描述草稿

活着吗不是效率工具，也不是打卡压力器。它只是给今天留一个很小的信号：我还活着。

你可以在每天开始或结束时，选一个心情，确认今天还在；写几句随笔，保存一张照片，记下今天最想做的三件事。所有这些会组成当天的电子日记，之后可以在打卡日历里回看。

如果你愿意，也可以添加好友。好友之间可以轻轻戳一下，问一句“还活着没？”对方回一句“我还活着”，状态就会更新。它不替代紧急联系、医疗或心理支持，只是一个轻量、温和、低压力的存在确认。

主要功能：

- 每日确认还活着
- 今日心情、天气氛围、随笔小记
- 最多 3 张照片记录当天
- 送给自己的一句话
- 今天最想做的三件事
- 打卡日历和历史电子日记
- 好友手机号添加、戳一下和“我还活着”回馈
- App 内二次确认注销账户

适合：

- 想给每天留下一个小小记录的人
- 想低压力记录心情和生活片段的人
- 想和少数好友保持轻量状态确认的人

## 隐私与权限说明

当前会涉及：

- 手机号：用于登录和好友手机号搜索；中国大陆手机号默认 `+86`，用户可直接输入 11 位号码。
- 昵称：用于好友展示。
- 日记/随笔/待办：用户主动输入并保存。
- 照片：用户主动从相册选择，保存到当天随笔日记。
- 好友关系和互动记录：用于好友申请、戳一下、我还活着回馈。

当前权限：

- 相册读取：用于添加随笔照片。
- 麦克风：不使用；打包前必须确认 Android 最终 Manifest 不包含麦克风权限。

## App Review 备注草稿

本应用是轻量每日确认和好友状态记录工具。用户可以通过手机号验证码登录，确认今天还活着，保存心情、随笔、照片、每日箴言和今天最想做的三件事，并通过好友手机号添加好友、戳一下和“我还活着”回馈进行轻量互动。

本应用不提供医疗、心理诊断、紧急救援或危机干预服务；好友互动仅用于轻量状态确认。

正式审核测试流程：

1. 使用手机号验证码登录。
2. 选择今日心情并确认还活着。
3. 保存随笔、照片、箴言和三件事。
4. 使用另一个手机号账号发送/接受好友申请。
5. 测试戳一下和“我还活着”回馈。

注意：开发测试账号只用于本地开发和内部 preview 包。正式生产包默认关闭测试账号入口，并通过 `npm run check:release-config` 检查。若审核团队无法接收中国大陆手机号验证码，请提示审核员点击登录页底部“先看演示模式”进入核心流程。
